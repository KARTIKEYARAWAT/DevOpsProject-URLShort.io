# React Dashboard (Frontend)

## Overview
A modern, responsive, and aesthetically pleasing web interface for the Distributed URL Shortener project. Built with **React** and **Vite**, it features real-time system monitoring, 3D visualizations, and a seamless user experience.

## Features
- **Real-time Status**: Visualizes the active/inactive state of storage nodes.
- **Glassmorphism UI**: Uses extensive backdrop blurs, gradients, and transparencies.
- **Source Code Preview**: Built-in viewer for project source code.
- **Documentation**: Embedded access to Architecture and Usage guides.

## Technology Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Project Structure
```
src/
```
